***快捷键***
| 快捷键 | 功能                                                   |
| ------ | ------------------------------------------------------ |
| DELETE | 删除选中控件及其子控件                                 |
| CTRL+C | 复制选中控件                                           |
| CTRL+V | 粘贴复制的控件到选中控件的子控件中或者粘贴到最外层控件 |

 ***工具栏***
 序号代表工具栏按钮顺序
 1. 设计窗口/切换显示边框
 2. 查看代码
 3. 清空布局
 4. 删除选中的控件
